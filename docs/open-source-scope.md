# Open Source Scope

This repository is the open-source toolkit part of the DuruNex ERP work.

It is not a dump of a private production ERP project.

## Included

- Documentation
- Sample workflows
- SELECT-only SQL report examples
- PHP helper examples
- Simple service examples
- ERP planning notes

## Not Included

- Private source code
- Customer records
- Bank data
- Invoice data
- Passwords
- API keys
- Server settings
- Commercial configuration files
- Production database dumps

## Reason

ERP projects often contain sensitive financial and operational information.

For this reason, only safe and reusable materials are shared here.
