# Security Policy

This repository is for documentation, helper code, and safe ERP examples.

## Do Not Share

Do not open an issue or pull request that includes:

- Database passwords
- API keys
- Server IP addresses
- Customer information
- Bank account information
- Real invoice or payment data
- Company-specific production configuration
- Personal data

## Reporting a Security Issue

If a security issue is found, please do not publish private details in a public issue.

Open a short issue saying that a security-related review is needed, without sharing secrets or private data.
